# Synthetic Examples

All files in this directory are synthetic examples. They are not real patient
data and should not be treated as clinical documentation.

- `synthetic_smartphrase_input.txt`: fake input text for parser/demo review.
- `synthetic_structured_output_summary.md`: expected structured interpretation summary.
- `synthetic_emr_note.txt`: expected clinician-facing EMR note shape.
- `synthetic_patient_roadmap.txt`: expected patient-facing roadmap shape.
